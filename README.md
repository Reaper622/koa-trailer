# koa-trailer
A trailer Website built with Koa:rocket:



## 项目技术栈

- 服务端：Node.js

- 服务端框架 : Koa2
- 转码器: Babel
- 爬虫引擎: puppeteer 
- 数据存储: MongoDB
- 数据存储对象模型工具: mongoose
- 数据加密: bcrypt
- 图床&云存储: 七牛云存储
- 工程化构建工具: Parcel (零配置webpack)

### 数据来源:

- 影视资讯：豆瓣API
